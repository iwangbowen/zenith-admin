# 表单设计

表单设计器用于配置流程发起时用户需要填写的表单。通过拖拽字段控件到画布，即可快速构建业务表单。

## 设计器布局

表单设计器采用三栏布局：

- **左侧控件面板**：可用字段类型列表，拖拽到中间画布即可添加
- **中间画布**：表单预览区，支持拖拽排序、嵌套布局
- **右侧属性面板**：选中字段后显示该字段的配置选项

## 支持的字段类型

表单字段共 **38 种**，左侧控件面板按以下分组展示。

### 基础控件

| 字段类型 | 名称 | 典型用途 |
| --- | --- | --- |
| `text` | 单行文本 | 标题、名称、简述 |
| `textarea` | 多行文本 | 详细说明、备注 |
| `number` | 数字 | 数量、不带货币的数值 |
| `amount` | 金额 | 带币种的金额，支持精度设置 |
| `date` | 日期 | 单个日期选择 |
| `dateRange` | 日期区间 | 起止日期选择 |
| `time` | 时间 | 时间选择 |

### 选择控件

| 字段类型 | 名称 | 典型用途 |
| --- | --- | --- |
| `select` | 下拉单选 | 单选场景 |
| `multiSelect` | 下拉多选 | 多选场景 |
| `autoComplete` | 自动完成 | 带建议项的输入 |
| `radio` | 单选框组 | 选项较少的单选 |
| `checkbox` | 复选框组 | 多选场景 |
| `switch` | 开关 | 是/否布尔值 |
| `slider` | 滑块 | 数值区间选择 |
| `tags` | 标签 | 多标签录入 |
| `colorPicker` | 颜色 | 颜色选择器 |

### 格式化控件

| 字段类型 | 名称 | 典型用途 |
| --- | --- | --- |

| `pinCode` | 验证码 | 定长验证码 / PIN 码 |

## 字段配置

选中画布上的字段后，右侧属性面板可配置：

**基础属性**：
- **字段名称**：显示在表单上的标签
- **字段标识**：表单数据的 key，建议英文命名，如 `leaveType`、`startDate`
- **是否必填**：勾选后该字段必须填写才能提交
- **占位提示**：输入框中的提示文字
- **帮助说明**：字段下方的补充提示
- **默认值**：字段初始值
- **栅格占宽 / 标签位置**：控制字段在表单中的布局宽度与标签对齐方式

**扩展属性**（按字段类型不同）：
- **选项值 / 选项联动**（select/multiSelect/radio/checkbox/autoComplete）：配置选项列表，支持按父字段值级联生成
- **长度 / 正则**（text/textarea/格式化控件）：最小/最大长度、正则校验及提示
- **精度 / 单位 / 范围**（number/amount/slider）：小数位数、单位、最小/最大值、步长
- **货币**（amount）：人民币 / 美元 / 欧元 / 英镑 / 日元
- **日期格式 / 时间格式**（date/dateRange/time）：如 `年-月-日`、`时:分:秒`
- **最大数量**（attachment/image/tags/pinCode）：允许上传的文件数或标签/验证码位数
- **评分上限**（rate）、**透明度**（colorPicker）、**省市区层级**（region）
- **多选**（userSelect/deptSelect/dictSelect）、**字典编码**（dictSelect）
- **公式表达式**（formula）：使用 `{字段key}` 引用其他字段
- **前缀**（serialNumber）：流水号前缀
- **可见性规则**：基于其他字段值（且/或组合）动态显示/隐藏该字段

## 布局能力

### 分栏（Row）

将字段拖拽到分栏组件中，可实现横向并排布局。分栏支持自定义每栏的宽度比例（如 6:6、4:8、3:9 等）。

### 分组（Group）

将相关字段放入分组中，可设置分组标题。分组可展开/收起，提升表单可读性。

### 分割线（Divider）

在不同区块之间添加视觉分隔线。

## 表单字段在流程中的使用

### 条件分支中使用

在流程设计的条件分支中，可以引用表单字段的值作为分支条件。例如：

```
如果 请假类型 == "病假" 且 天数 > 3
则走 "病假审批" 分支
否则走 "普通审批" 分支
```

### 节点表单权限

在审批/办理节点的配置中，可以设置每个表单字段的权限：

| 权限 | 说明 |
| --- | --- |
| 可读 | 审批人可以看到该字段的值，但不能修改 |
| 可编辑 | 审批人可以修改该字段的值 |
| 隐藏 | 审批人看不到该字段 |

例如：在"请假申请"流程中，发起人可以填写"请假原因"，而审批人只能看到但不能修改。

### 审批人指定中使用

在审批人节点配置中，可以使用表单字段来动态指定审批人：

- **表单内联系人**：选择表单中的人员字段作为审批人
- **表单内部门**：选择表单中的部门字段，自动取部门负责人作为审批人

## 设计规范建议

1. **字段标识使用英文**：便于后续在条件分支、审批人策略中引用
2. **合理分组**：将相关字段放入同一分组，提升表单可读性
3. **必填项标注**：关键信息设置为必填，避免提交后数据不完整
4. **预留说明字段**：在复杂表单中添加说明文字，引导用户填写
