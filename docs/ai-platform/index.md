# AI 能力

Zenith Admin 内置完整的 AI 能力平台：从多供应商接入、智能对话，到自定义智能体、知识库 RAG、工具生态与运营治理，对标主流大模型聊天与智能体应用的功能形态。

> AI 对话需要可用的系统服务商配置；未指定具体配置时使用启用的系统默认配置。允许用户自带 Key 时，也可以选择有效的个人配置。

## 能力总览

| 能力 | 当前实现 |
| --- | --- |
| 智能对话 | 多轮对话、SSE 流式输出、思维链展示、停止生成、重新生成、编辑后重发、会话导出、LLM 自动命名、会话分组与增量加载、对话标签 |
| 消息分支树 | 对齐 ChatGPT 的分支模型：重新生成 / 编辑重发创建兄弟分支，「‹ i/n ›」切换器回看任意分支 |
| 断线续传 | 生成与连接解耦（Redis 缓冲），刷新 / 断网后自动恢复实时输出，停止时保存已生成部分 |
| 自定义智能体 | 提示词 + 模型 + 知识库 + 工具组合预设；上架审核、智能体市场、一键克隆 |
| 多供应商 | OpenAI Compatible / Anthropic / Gemini 原生流式适配；一个配置多模型（自动发现）；能力标签（vision / tools / 上下文窗口） |
| 主备与并发 | 降级配置自动切换（failover 事件）；并发流上限（排队 15s） |
| Function Calling | 内置工具 + HTTP API 工具（管理员零代码注册），最多 5 轮循环，过程折叠卡片展示 |
| 图片理解 / 生成 | vision 模型图片输入（≤3 张）；`ai_image_model` 启用 generate_image 文生图工具 |
| 知识库 RAG | 文本 / txt / md / URL 网页入库，自动分块；pgvector SQL 向量检索（JS 余弦兜底）+ 混合加权；回答注入引用溯源 |
| 提示词模板 | 系统级 / 用户私有模板，`{{变量}}` 占位符，使用次数统计，历史版本快照与恢复 |
| 个人指令 | 用户级 Custom Instructions（关于我 / 回答风格），自动追加到所有对话 |
| 对话分享 | 只读分享链接（永久 / 7 天 / 30 天），免登录访问，按 IP 限流 |
| 模型对比 | Arena 双栏：同一提问并行发给两个模型流式对比，投票结果落库 |
| 模型评测 | 评测集 + 任务中心异步逐题运行，多模型回归对比 |
| 语音交互 | 浏览器 TTS 朗读回复、STT 语音输入 |
| 使用统计 | 对话 / 消息 / Token / 预估成本 / 首字延迟 / 成功率 / 活跃用户 / 模型分布 / 用户 Top 10 / 按日趋势 |
| 反馈闭环 | 点赞 / 点踩 + 原因，管理员筛选、上下文回放、处理备注、CSV 导出 |
| 审计与 Trace | 跨用户消息合规检索；生成调用链（检索 / 工具 / failover / LLM 轮次耗时）抽屉 |
| 安全合规 | API Key AES-256-GCM 加密、出站 SSRF 防护、敏感词过滤、接口限流与每日配额 |
| 用量配额 | `ai_daily_token_quota` 每用户每日 token 上限（0 = 不限制） |
| 聊天机器人 | Webhook 机器人属于即时通讯模块，详见 [即时通讯](../chat/index.md) |

## 快速上手

1. **配置服务商**：「AI 服务商」页新增配置（OpenAI 兼容网关最通用），填 API 地址 / Key / 模型，测试连接后设为默认并启用。
2. **开始对话**：「智能对话」页发送消息即可；推理模型自动展示思维链。
3. **进阶能力**（按需开启）：

| 目标 | 操作 |
| --- | --- |
| 函数调用 | 服务商配置勾选 `tools` 能力标签 |
| 图片理解 | 勾选 `vision` 能力标签 |
| 知识库问答 | 配置 `ai_embedding_model` → 建知识库传文档 → 对话挂载 |
| 自定义智能体 | 「智能体」页组合提示词 / 模型 / 知识库 / 工具 |
| 企业工具接入 | 「AI 工具」页注册 HTTP API |
| 用户自带 Key | 系统配置开启 `ai_allow_user_custom_key` |

## 系统配置速查

| 配置键 | 默认 | 说明 |
| --- | --- | --- |
| `ai_allow_user_custom_key` | `false` | 是否允许用户配置自己的 API Key |
| `ai_daily_token_quota` | `0` | 每用户每日 token 配额（0 = 不限制） |
| `ai_content_filter_enabled` | `false` | 输入侧敏感词过滤开关 |
| `ai_embedding_model` | 空 | 知识库向量化模型（空 = 关键词检索） |
| `ai_image_model` | 空 | 图片生成模型（空 = 关闭 generate_image 工具） |
