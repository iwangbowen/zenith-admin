# 产品概览

Zenith Admin 是一个面向企业后台、运营平台和内部工具场景的全栈后台管理系统。

## 核心定位

它的定位不是“做一个空白管理台壳子”，而是提供一套已经具备基础管理能力、并且适合继续扩展的项目底座。

适合这些团队：

- 想快速上线后台系统的创业团队
- 需要统一技术栈和代码规范的中小团队
- 需要做客户演示、PoC 或内部运营工具的项目组

## 技术栈

| 层级 | 技术 |
| --- | --- |
| 后端 | Hono v4 + Node.js |
| 前端 | React 19 + Vite 6 |
| UI | Semi Design v2 |
| ORM | Drizzle ORM |
| 数据库 | PostgreSQL |
| 校验 | Zod |
| 认证 | JWT Bearer Token |

## 产品价值

### 降低初始化成本

用户、角色、菜单、字典、文件管理等后台常见模块已经具备基础能力，不必每个项目都重头搭建。

### 保持演进空间

`shared` 层统一类型、常量和 schema，适合在新增业务时保持前后端一致性。

### 面向团队协作

接口规范、分页格式、页面布局、时间处理方式等都已经形成约定，适合多人协作开发。
